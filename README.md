# Customer-Complaint-Tracking-Group-16
This project, which is created by the members of Group #16, is a part of the **Complex Engineering Activity (CEA)** for the course **AICT (MCT-107L)**. In this project, the group has designed a _"Customer Complaint Tracking"_ system which involves the use of various ICT-based tools and uses _"Forecast Simulation"_ as the core technique for data analysis.
